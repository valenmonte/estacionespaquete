# Gráfico de temperatura media mensual por estación

Calcula la temperatura media mensual por estación y devuelve un gráfico
con las series de cada estación.

## Usage

``` r
grafico_temperatura_mensual(datos, titulo = "Temperatura")
```

## Arguments

- datos:

  data.frame con los datos.

- titulo:

  título del gráfico.

## Value

objeto ggplot.
